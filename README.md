# Bitcoin Power Insights

Aplicacion de analisis de trading de Bitcoin con Power Law Analysis y Trading Intradia.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Binance API (REST + WebSocket)

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Deploy

Deployed on Vercel. Push to `main` to auto-deploy.
